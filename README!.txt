Instructions for Compiling the Project:-

1.	Import the complete folder "Project_531_Final_Megha&Nikhil" into Eclipse

2.	In Eclipse, run Compiler.java from src/Triangle

3.	Now, go to Run >> Run Configurations...

3.	In the Main tab, 	for Project select the said project (Project_531_Final_Megha&Nikhil) &
						in the Main class give Triangle.Compiler

4.	In the Arguments tab, for Program arguments write name of a test case (ex. repeatTestCase.tri)

5.	Hit Run and see the result on Console.

-----------------------------------
NOTE:	The above steps need to be done for each test case.
		The relevant Test Cases (Required and Extra) have been provided in the main project folder.
